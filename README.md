# comp-vision-crowd-density-map

## Setup

Notebook calistirmadan once ayni Python ortamina bagimliliklari kurun:

```bash
/opt/homebrew/bin/python3 -m pip install -r requirements.txt
```

Notebook kernel hatasi alirsaniz `ipykernel` ayni ortama kurulu olmalidir. Bu repo artik `requirements.txt` icinde onu da listeliyor.
